# PostgreSQL & fio Performance Testing Suite

## Архитектура стенда
- **Управляющая машина**: Control host (MacBook любой другой хост с набором скриптов управления процессом тестирования)
- **Тестовые машины**: от 1 до 4 виртуальных машин машины под управлением ОС Debian 12, PostgreSQL 17, ФС btrfs, которые отвечают требованиям в `TESTING_METHOD.md`
- **Сценарии**: Тест fio → Тест pgbench → сбор метрик → анализ и визуализация результатов

## Быстрый старт
1. Настройка тестовых виртуальных машин согласно `TESTING_METHOD.md`
2. На все тестовые виртуальные машины необходимо скопировать скрипты из `scripts/`
3. На control host производится сбор и анализ полученных результатов тестирования:  
   ```bash
   ./control/run_tests.sh
   python3 control/tools/parse_results.py results/run_20251106_1200
